# Landing Page - Peixaria Rio Paraguai

## Descrição
Landing page profissional desenvolvida para a Peixaria Rio Paraguai, utilizando HTML5, CSS3 e JavaScript vanilla. O design é totalmente responsivo e otimizado para conversão.

## Paleta de Cores
- **Azul Náutico (#004e64)**: Cor principal para profissionalismo
- **Vermelho Coral (#e5383b)**: Destaque para botões e ofertas urgentes
- **Azul Água Clara (#a0ced9)**: Cor secundária para leveza
- **Branco Areia (#fcfaf9)**: Fundo principal, suave e limpo

## Como Usar

### 1. Hospedagem Local
Para visualizar a landing page localmente:
1. Abra o arquivo `index.html` em qualquer navegador moderno
2. Todos os arquivos devem estar na mesma estrutura de pastas

### 2. Hospedagem Web
Para hospedar online:
1. Faça upload de todos os arquivos para seu servidor web
2. Certifique-se de manter a estrutura de pastas
3. Configure o domínio para apontar para o arquivo `index.html`

### 3. Personalização
Para personalizar o conteúdo:

#### Alterando Textos:
- Edite o arquivo `index.html`
- Localize as seções desejadas e modifique o conteúdo

#### Alterando Cores:
- Edite o arquivo `styles.css`
- Modifique as variáveis CSS no início do arquivo:
```css
:root {
    --azul-nautico: #004e64;
    --vermelho-coral: #e5383b;
    --azul-agua-clara: #a0ced9;
    --branco-areia: #fcfaf9;
}
```

#### Alterando Imagens:
- Substitua as imagens na pasta `images/`
- Mantenha os mesmos nomes de arquivo ou atualize as referências no HTML

#### Alterando Informações de Contato:
- Edite os telefones, endereço e email no arquivo `index.html`
- Atualize os links do WhatsApp com os números corretos
